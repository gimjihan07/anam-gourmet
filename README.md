# 🗺️ Anam Gourmet (안암 고메)

> **고려대학교(안암) 주변의 숨은 맛집과 추천 식당을 한눈에 확인할 수 있는 웹 서비스입니다.**

## 📝 프로젝트 소개
Anam Gourmet는 대학생들과 지역 주민들이 안암역 주변의 식당 정보를 효율적으로 탐색하고 공유할 수 있도록 제작한 웹 프로젝트입니다. 
가성비 좋은 식당부터 혼밥하기 좋은 곳, 모임 장소까지 사용자의 목적에 맞는 맞춤형 맛집 정보를 제공하는 것을 목표로 합니다.

## 🎯 프로젝트 목표
* 안암(고려대) 주변 상권 활성화 및 숨은 맛집 발굴
* 사용자 목적(혼밥, 가성비, 모임 등)에 맞는 빠른 식당 탐색 구현
* 직관적이고 깔끔한 UI를 통한 편리한 사용자 경험 제공

## ✨ 주요 기능

### 🔍 맛집 카테고리 분류
* 한식, 중식, 일식, 양식, 세계요리, 카페/디저트 등 카테고리별 조회

### 💡 맞춤형 큐레이션 리스트
* 혼밥하기 좋은 곳 / 가성비 최고 맛집 / 회식 및 모임 장소 추천 테마 제공

### 🗺️ 위치 및 상세 정보 제공
* 식당의 주요 메뉴, 가격대, 위치 정보 확인 기능

## 🛠️ 사용 기술

### Front-end
* HTML5
* CSS3
* JavaScript (Vanilla JS)

### Deployment
* GitHub Pages

## 📥 입력 / 출력 (Input / Output)

### 입력 (Input)
* 사용자의 카테고리 선택 및 필터 클릭
* 특정 맛집 검색어 입력

### 출력 (Output)
* 조건에 맞는 맛집 목록 및 상세 카드 표시
* 식당 위치, 메뉴, 가성비 태그 등의 정보 시각화
